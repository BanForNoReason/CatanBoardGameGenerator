Online Catan Board Generator for Fair, Balanced Maps

This Catan board generator creates fast, fair layouts for 3–4 player and 5–6 player games. It spreads high-probability numbers (6s and 8s), balances resources, and avoids brutal starter spots so everyone gets a fun game.

How to Use the Catan Map Generator

  1. Select Standard (3–4 players) or Expanded (5–6 players).
  2. Choose options like Block adjacent 6 & 8 or Block same resources.
  3. Click Generate until you find a layout your group loves.

The generator keeps the official Catan tile counts and number tokens, while enforcing your chosen balance rules.

Note for 5–6 player expansion: If you enable all constraint options simultaneously (especially Block same resources), the generator may struggle to create a valid board due to the high number of restrictions. If this happens, try disabling one or more options to allow more flexibility in board generation.

Catan Board Generator FAQ

Does this follow official Settlers of Catan rules?

Yes. Tile and number distributions match the base game and 5–6 player expansion; the options only change how they are arranged on the board.

Can 6s and 8s ever touch?

If Block adjacent 6 & 8 is checked, the generator keeps high-probability numbers separated for a more balanced map.
